# Unit Converter Tool

A simple Python-based Unit Converter Tool that allows users to convert values between different units.

## Features

- Length conversion (meters, kilometers, centimeters, etc.)
- Weight conversion (grams, kilograms, pounds, etc.)
- Easy-to-use command-line interface
- Fast and accurate calculations

## Requirements

- Python 3.x

## How to Run

1. Clone the repository:

```bash
git clone https://github.com/megana-anand/Unit-Convertor-Tool.git
```

2. Navigate to the project folder:

```bash
cd Unit-Convertor-Tool
```

3. Run the program:

```bash
python main.py
```

## Example

```text
Enter value: 100
Select conversion type: Meters to Kilometers
Result: 0.1 Kilometers
```
